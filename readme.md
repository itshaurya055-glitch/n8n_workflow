## 📥 Importing a Workflow

1. Download the desired `workflow.json`.
2. Open your n8n workspace.
3. Click **Import from File**.
4. Select the downloaded workflow.
5. Configure the required credentials.
6. Execute the workflow.

---

## 🔐 Credentials

Some workflows require external services.

Examples include:

* OpenAI API
* Groq API
* Gemini API
* GitHub Personal Access Token
* Gmail OAuth
* Slack
* Discord
* Google Sheets

Create your own credentials inside n8n before running a workflow.
